# Advent of Code 2024 solver

## Run solver for day 1

The input is expected to be in a text file called `01.in`.

Run tests of _day 1_:

```bash
./aoc.py 1 -t
```

Solve _day 1_, part 1:

```bash
./aoc.py 1 -p1
```

Solve _day 1_, part 2:

```bash
./aoc.py 1 -p2
```

Solve _day 1_, both parts:

```bash
./aoc.py 1
```
